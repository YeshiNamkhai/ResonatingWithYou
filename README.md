# Resonating With You
Welcome to repository of **Resonating With You**, an event held in Febrary 2026 at Dzamling Gar featuring an immersive sound experience designed to help you better understand yourself and the characteristics of your own perception.

## Requirements
To run the scripts you need a working knowledge of Python programming language, some DSP and audio programming and  knowledge MIDI devices such as Novation Launchpad. Create a custom environement for Python 3.11 and add module from the [requirements](requirements.txt).

### Novation Launchpad
- MK1 
- MK2
